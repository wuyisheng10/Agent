Google Sheets 初始化步驟
1. Google Cloud Console → 建立 Service Account
   啟用 Google Sheets API + Google Drive API
2. 下載 JSON 金鑰 → 存為 config/google_credentials.json
3. 建立 Google Sheets 試算表，新增三張工作表：
   市場名單 / 夥伴名冊 / 激勵紀錄
4. 將試算表共用給 Service Account email（編輯者）
5. 複製試算表 ID → 填入 config/settings.json
   # URL: docs.google.com/spreadsheets/d/[這段ID]/edit
6. 安裝相依套件：pip install gspread google-auth
7. 執行排程設定：powershell -ExecutionPolicy Bypass -File setup_scheduler.ps1
