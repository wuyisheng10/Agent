# 群雁培訓系統 Prompt / Skill 分工

## Skill / 流程型
這些功能應該由 agent 或固定流程負責，不應只靠 prompt。

- 培訓模組建立與查詢
- 培訓課程建立與查詢
- 培訓反思寫入與進度聚合
- 七天法則啟動與每日回報
- 課後行動建立與回報
- 個人培訓進度與總表查詢

## Prompt 型
這些功能主要是生成內容，適合透過 prompt 調整文字品質。

- 課後反思摘要優化
- 教練回饋建議
- 七天法則每日鼓勵文案
- 課後行動提醒話術
- 培訓心得整理文案

## 混合型
先用 skill 判斷資料與步驟，再用 prompt 生成文字。

- 培訓進度建議下一步
- 根據七天法則紀錄產生教練提醒
- 根據課後行動狀態產生 follow-up 建議
- 培訓總表的 AI 重點摘要

## 建議實作順序
1. 先把模組、課程、反思、七天法則、課後行動做成穩定 skill。
2. 再把教練建議、鼓勵話術、總表摘要做成 prompt。
3. 最後才把進度分析、階段判斷、推薦下一步做成混合型 AI 功能。
