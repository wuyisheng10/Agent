# Qunyan Training Phase 1 Commands and Web Plan

這份文件定義第一期培訓系統在：

- LINE 指令
- `5168` 執行選單
- 網頁按鈕
- 網頁表單

上的操作入口。

對應文件：

- `docs/qunyan_training_system_plan.md`
- `docs/qunyan_training_phase1_data_model.md`

## 一、第一期要先上線的功能

第一期只做這 5 類：

1. 新增培訓模組
2. 新增培訓課程/場次
3. 填寫 `悟到 / 學到 / 做到 / 目標`
4. 查詢個人培訓進度
5. 查詢團隊培訓總表

## 二、LINE 指令規劃

## 1. 新增培訓模組

### 指令

```text
新增培訓模組 模組名稱 | 模組類型 | 學習目標 | 核心摘要
```

### 範例

```text
新增培訓模組 四個勇於 | 領導人特質 | 建立領導人思維與承擔觀念 | 勇於學習、勇於認錯、勇於改變、勇於承擔
```

## 2. 新增培訓課程

### 指令

```text
新增培訓課程 課程名稱 | 模組名稱 | 日期 | 時間 | 地點 | 講師 | 對象
```

### 範例

```text
新增培訓課程 領導人特質｜四個勇於 | 四個勇於 | 2026-04-10 | 19:30 | 台南教室 | 鐘老師 | 夥伴
```

## 3. 填寫課後反思

### 指令

```text
新增培訓反思 姓名 | 課程名稱 | 悟到 | 學到 | 做到 | 目標
```

### 範例

```text
新增培訓反思 王小美 | 領導人特質｜四個勇於 | 我發現自己常不願認錯 | 四個勇於是連動的 | 這週每天回報市場 | 本月建立穩定回報習慣
```

## 4. 查詢個人培訓進度

### 指令

```text
查詢培訓進度 姓名
```

### 範例

```text
查詢培訓進度 王小美
```

## 5. 查詢團隊培訓總表

### 指令

```text
查詢培訓總表
```

## 6. 查詢培訓模組

### 指令

```text
查詢培訓模組
查詢培訓模組 模組名稱
```

## 7. 查詢培訓課程

### 指令

```text
查詢培訓課程
查詢培訓課程 日期
查詢培訓課程 模組名稱
```

## 三、建議加入 5168 執行選單

可新增一個新群組：

## 培訓系統 2.0

建議編號可在現有編號後追加。

### 建議項目

- `新增培訓模組`
- `查詢培訓模組`
- `新增培訓課程`
- `查詢培訓課程`
- `新增培訓反思`
- `查詢培訓進度`
- `查詢培訓總表`

### 建議呈現方式

```text
培訓系統 2.0
 79. 新增培訓模組
 80. 查詢培訓模組 ▶
 81. 新增培訓課程
 82. 查詢培訓課程 ▶
 83. 新增培訓反思
 84. 查詢培訓進度 ▶
 85. 查詢培訓總表 ▶
```

如果後面已有其他功能佔用 `79` 之後的號碼，就直接順延即可，原則是同群組集中。

- 對象

## 六、網頁回應格式建議

## 1. 查詢培訓模組

回應應顯示：

- 模組名稱
- 類型
- 學習目標
- 核心摘要

## 2. 查詢培訓課程

回應應顯示：

- 課程名稱
- 對應模組
- 日期時間
- 地點
- 講師
- 對象

## 3. 查詢培訓進度

回應應顯示：

- 已完成模組
- 最近上的課
- 最近一次反思
- 最近一次做到
- 下一個建議模組

## 4. 查詢培訓總表

回應應顯示：

- 每個人完成的模組數
- 最近是否有填反思
- 最近是否有行動目標
- 哪些人需跟進
