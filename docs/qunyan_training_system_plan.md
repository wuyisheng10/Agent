# 群雁培訓系統規劃

## 目標
根據 `悟到 / 學到 / 做到 / 目標` 教材，把培訓從單純課後整理，擴充成可追蹤、可回報、可帶線的培訓系統。

## 核心模組
- 培訓模組中心：四個勇於、七天法則、2689、市場實戰等主題模組化。
- 培訓課程管理：每次開課都有日期、講師、地點、對象與對應模組。
- 培訓反思系統：每堂課後記錄悟到、學到、做到、目標。
- 培訓進度追蹤：看個人完成哪些模組、最近上了什麼課、下一步推薦什麼。
- 七天法則追蹤：新人前七天的陪跑與日回報。
- 課後行動回報：把培訓內容轉成具體行動與回報。

## 和現有系統的整合
- 夥伴系統：培訓反思、七天法則、課後行動都以夥伴為主體。
- 行事曆：培訓課程可對應未來活動與提醒。
- 歸檔系統：教材、逐字稿、課後資料可歸進培訓資料。
- AI prompt / skill：只負責生成文字品質，不負責流程狀態。
